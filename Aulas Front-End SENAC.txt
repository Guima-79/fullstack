02/06

GIT =  Controle de versionamento local.

GIT config -- Global user.name "seu nome" =  Configura conta no GIT
GIT config -- Global user.email "email" = Configura e-mail no GIT
GIT init = Cria pasta vazia para o repositório GIT
GIT add <Nome do arquivo> 01 = Envia o arquivo para o purgatório para analise de versionamento
GIT add -A = Manda todos os arquivos para analise que antecede a pasta GIT
GIT commit - m + informação da alteração = Envia versão para pasta GIT
GIT status = Mostra 
GIT add -u =  Envia a versão mais nova do arquivo para o repositório GIT
GIT push -u origin main  Manda atualização do GIT local para o GITHUB
Git commit -m 'texto' = Grava no GIT local
Git push -u origin main = Salva no GIT HUB

GIT HUB

Git branch -M main = Transformar o GIT em um repositório main
